# Neeh brand assets

This directory contains generated SVG assets for the Neeh SDK. The canonical 
design specification and generator are defined in `BRAND.md` and 
`brand/render_logo.py`. Do not redraw or independently edit these assets.

| File | Use |
| --- | --- |
| `neeh-icon.svg` | Paper-background icon for SDK documentation |
| `neeh-logomark.svg` | Primary Amber mark on light or transparent surfaces |
| `neeh-logomark-on-ink.svg` | Reversed mark for dark Ink surfaces |

Only these documentation assets are included in the SDK repository.

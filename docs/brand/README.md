# Neeh brand assets

This directory contains generated SVG assets for the Neeh SDK. The canonical 
design specification and generator are defined in `BRAND.md` and 
`brand/render_logo.py`. Do not redraw or independently edit these assets.

| File | Use |
| --- | --- |
| `neeh-icon.svg` | Paper-background icon for SDK documentation |
| `neeh-logomark.svg` | Primary Amber mark on light or transparent surfaces |
| `neeh-logomark-on-ink.svg` | Reversed mark for dark Ink surfaces |
| `neeh-logomark-on-ink.png` | Rasterized reversed mark for bitmap contexts |
| `neeh-social-preview.svg` | Source for the 2:1 GitHub social preview |
| `neeh-social-preview.png` | Raster social preview uploaded to GitHub |

Only these documentation assets are included in the SDK repository.

## Visual direction

- Use flat Ink, Paper, and Amber color fields. Do not add gradients, glows, or
  generic floating geometry.
- Use the canonical generated mark without decorative containers or effects.
- Prefer clear typography, negative space, and real product evidence over
  ornamental technology imagery.
- Keep every element functional: identity, product meaning, or verified
  technical context.
